## 🚀 INSTRUCCIONES PARA PUBLICAR

### Opción 1: Hosting compartido (recomendado para empezar)
1. Contrata hosting en **Raiola Networks**, **SiteGround** o **Webempresa** (los mejores en España)
2. Sube toda la carpeta `/mi-web-emprender/` por FTP (usa FileZilla) a la carpeta `public_html/`
3. Configura tu dominio para que apunte al hosting
4. Activa el certificado SSL gratuito (Let's Encrypt) en el panel de control

### Opción 2: Netlify (gratis para empezar)
1. Crea cuenta en [netlify.com](https://netlify.com)
2. Arrastra y suelta la carpeta completa en el dashboard
3. Netlify te dará una URL automática tipo `tusite.netlify.app`
4. Puedes conectar tu dominio propio gratuitamente

### Opción 3: GitHub Pages (gratis, requiere conocimiento técnico)
1. Crea repositorio en GitHub
2. Sube los archivos con `git push`
3. Activa GitHub Pages en Settings → Pages
4. URL: `tuusuario.github.io/mi-web-emprender`

### Pasos post-publicación
1. **Google Search Console**: Ve a [search.google.com/search-console](https://search.google.com/search-console), verifica tu dominio y envía el sitemap.xml
2. **Google Analytics**: Crea cuenta en [analytics.google.com](https://analytics.google.com), copia el ID (G-XXXXXXXX) y descomenta el script en el `<head>` de `index.html`
3. **Reemplaza la URL base**: Busca y reemplaza `https://www.emprendeenespana.es` por tu dominio real en todos los archivos HTML y en `sitemap.xml`
4. **Textos legales**: Completa las páginas `aviso-legal.html`, `politica-privacidad.html` y `politica-cookies.html` con tus datos reales
